🎯 OBJECTIF
───────────
Permettre aux citoyens de demander l'apposition d'une mention marginale sur un
acte d'état civil existant (divorce, reconnaissance, adoption, etc.)

🚀 INSTALLATION RAPIDE
──────────────────────

1. Exécuter SQL
   └─ Supabase → SQL Editor → create-avis-mentions.sql

2. Créer bucket Storage
   └─ Supabase → Storage → Créer "documents" (public)

3. Vérifier mairies
   └─ UPDATE mairies SET code_mairie = UPPER(SUBSTRING(ville, 1, 3))

4. Pousser le code
   └─ git add . && git commit -m "Module Avis Mention" && git push

5. Tester
   └─ /citoyen/avis-mention
