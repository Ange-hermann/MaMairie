## 📋 **ÉTAPES EXACTES**

### **1. Allez sur Supabase**
```
https://supabase.com
→ Connectez-vous
→ Sélectionnez votre projet
```

### **2. Ouvrez SQL Editor**
```
Dans le menu à gauche :
→ Cliquez sur "SQL Editor"
→ Cliquez sur "New query"
```

### **3. Copiez le script**
```
Ouvrez le fichier : FORCER-MISE-A-JOUR.sql
→ Sélectionnez TOUT (Ctrl+A)
→ Copiez (Ctrl+C)
```

### **4. Collez dans Supabase**
```
Dans SQL Editor de Supabase :
→ Collez (Ctrl+V)
→ Cliquez "Run" (en bas à droite)
```

### **5. Vérifiez le résultat**
```
Vous devriez voir :
✅ "FONCTION MISE À JOUR !"
```
