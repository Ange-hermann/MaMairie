# Dossier Public - MaMairie

Ce dossier contient les fichiers statiques accessibles publiquement.

## 📁 Structure

```
public/
├── images/              # Images et illustrations
│   ├── logo/           # Logos de l'application
│   ├── icons/          # Icônes
│   └── backgrounds/    # Images de fond
├── documents/          # Documents PDF exemples
├── fonts/              # Polices personnalisées (optionnel)
├── favicon.ico         # Icône du site
├── manifest.json       # Manifest PWA
└── robots.txt          # Configuration SEO
```

## 🖼️ Images à Ajouter

### Logos
- `logo-192.png` - Logo 192x192px (pour PWA)
- `logo-512.png` - Logo 512x512px (pour PWA)
- `logo-full.png` - Logo complet avec texte
- `logo-icon.png` - Icône seule

### Partenaires

### Illustrations
- `hero-image.png` - Image principale de la page d'accueil
- `dashboard-preview.png` - Aperçu du dashboard
- `mobile-app.png` - Aperçu de l'app mobile

## 📄 Documents

### Exemples de PDF
- `extrait-naissance-exemple.pdf` - Exemple d'extrait de naissance
- `guide-utilisation.pdf` - Guide d'utilisation
- `conditions-utilisation.pdf` - Conditions d'utilisation

## 🎨 Recommandations

### Format des Images
- **Logos** : PNG avec fond transparent
- **Photos** : JPG optimisées (< 200KB)
- **Icônes** : SVG ou PNG

### Tailles Recommandées
- Logo principal : 400x100px
- Icône : 64x64px
- Favicon : 32x32px
- Images hero : 1920x1080px

## 🚀 Utilisation

Les fichiers dans ce dossier sont accessibles via :
```
http://localhost:3000/nom-du-fichier.ext
```

Exemple :
```
http://localhost:3000/logo-full.png
```

- Utilisez des noms de fichiers descriptifs
